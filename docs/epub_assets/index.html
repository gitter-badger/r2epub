CSS and SVG files adapted for EPUB usage

The files in this directory are the (slightly) modified versions of some original W3C files, adapted for EPUB:

base.css
A modified version of the W3C base.css file, with all references to the TOC removed, text margins changed, and some page control statements added.
A collection of SVG files (vertical bars on the starting page, logo files, etc.)
XML files in an EPUB file are not supposed to use XML entities; however, the original versions of these SVG files refer to the SVG DTD which, on its turn, use XML modularization, i.e., lots of XML entities. The SVG files in this collection are identical to the original ones, except that the DTD references are removed.
Strictly speaking this is not necessary insofar as EPUB readers can usually handle these SVG files, but epubcheck does complain about this. Note that there is an open discussion in the epubcheck project whether such entities should not be allowed in DTD-s which are usually ignored by reading systems anyway...
