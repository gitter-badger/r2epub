r2epub — Converting HTML documents produced by Respec into EPUB 3.2

Typescript program to convert W3C documents, produced by ReSpec, to EPUB 3.2. The conversion can be done via:

  • A simple command line interface
  • Using a Web Server API (deployed through the online service running on heroku)
  • Using a client side processing version (deployed through a separate conversion page on this site)

See the full documentation for the details and usage examples.
