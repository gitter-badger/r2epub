Interface Arguments

CLI arguments used by the create_epub entry function.

The original content file may have to be run through the W3C spec generator service before further processing. If that is the case (see Arguments.respec), it is also possible to set some of the ReSpec configuration options, overwriting the values set in the config entry of the original file (see Arguments.config). See the ReSpec editor's guide for details of the values.
